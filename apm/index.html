Azzurra Proxy Monitor

Azzurra Proxy Monitor (APM) è un proxy monitor progettato per il controllo degli utenti di un singolo server.

Pur essendo progettato per distribuire il carico di lavoro del proxy monitor centralizzato di Azzurra, APM è comunque in grado di funzionare autonomamente, impostando K:line sul server a cui è connesso nel caso rilevi un proxy.
